c. Variables to Show Relationships Between Quantities:

We have also seen variables being used to represent relationships between two quantities.

For example, we can express how to change feet into inches using words:

To find the number of inches in a specified number of feet, multiply the number of feet by 12.

We can also show this relationship using letters and symbols:

\(i = 12f\)

In this case, we may treat one of the variables as an unknown.

For example, we may ask, "+How many inches are in 3 feet?"

In this case, i is an unknown number and we can determine that it is equal to 36.

Alternatively, we can treat the variables as a set of numbers and examine how one variable changes as the other changes.

I may want to know how many inches are in 1 foot, 2 feet, 3 feet, 4 feet, and 5 feet.
