b. Variables to Communicate Mathematical Ideas:

You have also seen letters used to communicate mathematical ideas, such as the Properties of Addition and Multiplication.

For example, we can state the Commutative Property of Addition using words:

Commutative Property of Addition: Changing the order of the addends in an addition problem does not change the result.

We can also illustrate the Commutative Property of Addition using letters and symbols:

\(Commutative Property of Addition: a + b = b + a\)

We also see letters in geometry formulas.

For example, we can state the formula for the area of a triangle using words:

Area of a Triangle: Multiply the base and the height of the triangle and then divide the result by two.

We can also show this formula using letters and symbols:

\(A = \frac{b \times h}{2}\)
